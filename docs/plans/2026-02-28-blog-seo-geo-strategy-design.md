# 코리너스 블로그 SEO/GEO 전략 설계

**작성일:** 2026-02-28
**상태:** 설계 승인 완료

---

## 배경

코리너스(koreaners.co) 블로그를 SEO + GEO 채널로 활용해, 한-일 크로스보더 관련 업계 관계자가 검색/AI 질문 시 코리너스가 노출되게 만드는 전략.

- **현재 상태:** 블로그 시스템 완비 (Next.js + Supabase, 관리자 에디터, 다국어 지원), 발행 글 거의 없음
- **타겟 독자:** 한-일 크로스보더 모든 업계 관계자 (마케터, 대표, 에이전시, 관광/팬 타겟 업체 등)
- **목표:** SEO (서비스+정보성 키워드) + GEO (ChatGPT, Perplexity, Google AI Overview 노출)
- **콘텐츠 생산:** AI 주도 + 사람 감수
- **언어:** 한국어 우선, 일본어 나중에

## 접근법: 하이브리드 "SEO 라이브러리 + GEO 트렌드"

- Phase 1 (기초): 에버그린 콘텐츠 30-40개 빠르게 생산 → SEO 기반 확보
- Phase 2 (정기): 주 1-2개 트렌드/뉴스 + FAQ/데이터 구조화 → GEO 최적화
- Phase 3 (확장): 성과 분석 후 일본어 버전 + 고성과 키워드 심화

## Section 1: 필러 토픽 & 키워드 클러스터

### 6개 필러

| # | 필러 토픽 | 우선순위 | 이유 |
|---|-----------|----------|------|
| 1 | 일본 인플루언서 마케팅 | 최우선 | 코리너스 핵심 서비스, 롱테일 풍부 |
| 2 | 일본 체험단/시딩 | 최우선 | 경쟁 거의 없음 + 코리너스 핵심 서비스 |

| 4 | 일본 SNS 플랫폼 가이드 | 높음 | 채널별 차이 궁금해하는 마케터 유입 |

### 클러스터 예시

**필러 1: 일본 인플루언서 마케팅**
- 일본 인플루언서 마케팅 비용 가이드 (2026)
- 일본 인스타그램 vs 유튜브 vs 틱톡 인플루언서 비교
- 일본 마이크로 인플루언서 섭외 5단계 프로세스
- 한국 브랜드 일본 인플루언서 성공 사례 3선
- 일본 인플루언서 마케팅 ROI 측정법
- 일본 KOL이란? 인플루언서와의 차이
- 일본 인플루언서 계약 시 주의사항

**필러 2: 일본 체험단/시딩**
- 일본 체험단 마케팅이란? 한국과의 차이점
- 일본 리뷰 체험단 운영 A to Z
- 큐텐 vs 라쿠텐 체험단 전략 비교
- 일본 대량 시딩 캠페인 설계 방법
- 일본 체험단 비용 및 기대 효과
- 일본 아마존 리뷰 마케팅 가이드

### 경쟁 분석

- **어센트 코리아**: "일본 마케팅 대행사" 장악 — 정면 승부 피하고 인플루언서/체험단 롱테일로 우회
- **공백 키워드**: 일본 체험단, 일본 관광객 인플루언서 마케팅, 한일 크로스보더 마케팅
- **차별화**: 양방향(한→일 진출 + 방일 관광객 마케팅) 콘텐츠는 현재 시장에 부재

## Section 2: 콘텐츠 템플릿 — GEO 최적화 구조

### 글 구조 (모든 글 공통)

1. **Quick Answer (40-80자)** — AI 인용의 44%가 도입부에서 발생
2. **핵심 요약 박스** — bullet 3-5개, 수치 포함
3. **본문 (H2/H3 의문문 헤딩)** — 각 H2 아래 300-500자 + 표/수치
4. **비교표/데이터 테이블** — AI가 표 형식 데이터 선호
5. **FAQ 섹션 (3-5개)** — Google AI Overview 노출 3.2배 상승
6. **CTA** — 코리너스 문의 유도

### GEO 최적화 규칙

| 규칙 | 이유 |
|------|------|
| 첫 문단에 직접 답변 | AI 인용의 44%가 도입부 30%에 집중 |
| H2는 의문문 | AI가 Q&A 단위로 파싱 |
| 모든 글에 수치/통계 최소 3개 | 통계 콘텐츠가 AI 인용률 최상위 |
| 비교표 최소 1개 | 표 형식은 AI가 답변에 활용 |
| FAQ Schema 마크업 | Google AI Overview + Perplexity 노출 유리 |
| 글 길이 1,500-3,000자 | SEO 충분 + AI가 핵심 파악 가능한 범위 |

### Schema Markup 추가 (기술)

현재 BlogPosting JSON-LD에 추가:
- **FAQPage Schema** — 모든 글 FAQ 섹션
- **Author Schema** — 저자 정보 (E-E-A-T)
- **Organization Schema** — 코리너스 회사 정보 강화

## Section 3: 콘텐츠 생산 파이프라인

### 워크플로우

1. **키워드 선정** — 월 1회, 10-15개 배치
2. **AI 초안 생성** — 키워드 + 템플릿 → Claude/ChatGPT (1글 15-30분)
3. **사람 감수** — 팩트체크, 코리너스 인사이트 추가, CTA 확인 (1글 20-40분)
4. **발행** — 관리자 에디터 업로드, meta_title/description/썸네일 세팅
5. **검증** — Google Search Console 인덱싱 확인, 1-2주 후 모니터링

### 발행 페이스

| Phase | 기간 | 목표 | 내용 |
|-------|------|------|------|
| Phase 1 (기초) | 1-2개월 | 30-40개 | 에버그린 콘텐츠 집중 (필러별 5-7개) |
| Phase 2 (정기) | 3개월~ | 주 1-2개 | 트렌드/뉴스 + 에버그린 혼합 |
| Phase 3 (확장) | 6개월~ | 성과 기반 | 일본어 버전, 고성과 키워드 심화 |

### Phase 1 배분 (첫 30개)

| 필러 | 글 수 | 우선 키워드 |
|------|-------|------------|
| 일본 인플루언서 마케팅 | 7개 | 비용, 섭외법, ROI, 성공사례, KOL 정의 |
| 일본 체험단/시딩 | 6개 | 체험단이란, 운영법, 플랫폼 비교, 시딩 설계 |
| K-뷰티/K-컬처 일본 | 5개 | 트렌드, 드럭스토어 입점, MZ세대, 성공사례 |
| 일본 SNS 플랫폼 | 5개 | 인스타/틱톡/X/LINE 각각 + 전체 비교 |
| 일본 관광객 마케팅 | 4개 | 인바운드란, 인플루언서 활용, SNS전략 |
| 크로스보더 실무 | 3개 | 크로스보더란, 대행사 선택 기준, 비용 가이드 |

### 감수자 체크리스트

- [ ] 수치/통계에 출처가 있는가?
- [ ] 코리너스 고유 인사이트가 1개 이상 포함되었는가?
- [ ] Quick Answer가 40-80자로 핵심을 담고 있는가?
- [ ] FAQ가 실제 고객이 물어볼 만한 질문인가?
- [ ] CTA가 자연스럽게 연결되는가?

## Section 4: 성과 측정 & 반복 전략

### SEO KPIs

| 지표 | 측정 도구 | Phase 1 목표 (2개월) | Phase 2 목표 (6개월) |
|------|-----------|---------------------|---------------------|
| 인덱싱된 페이지 수 | Search Console | 30개+ | 60개+ |
| 노출 수 | Search Console | 월 5,000+ | 월 30,000+ |
| 클릭 수 | Search Console | 월 200+ | 월 2,000+ |
| 평균 게재순위 | Search Console | 상위 30위 10개+ | 상위 10위 5개+ |
| 블로그→문의 전환 | GA4 + Supabase | 월 2건+ | 월 10건+ |

### GEO KPIs

| 지표 | 측정 방법 | 목표 |
|------|-----------|------|
| AI 레퍼럴 트래픽 | GA4 source 필터 (chatgpt.com, perplexity.ai) | 트래킹 시작 → 증가 추세 |
| AI Overview 노출 | Search Console "검색 노출 형태" | 월 1개+ 키워드 |
| 브랜드 멘션 | Perplexity/ChatGPT 수동 질문 | 분기 1회 |

### 반복 사이클 (매월)

- 고성과 글 → 같은 필러에서 클러스터 확장
- 저성과 글 → 제목/Quick Answer 리라이트
- Search Console "쿼리" 탭 → 글 없는 유입 키워드 발견 → 새 글
- 경쟁사 모니터링 → 빈틈 체크

### Phase 전환 기준

- Phase 1→2: 30개+ 발행, 인덱싱 확인, 5개 키워드 상위 30위
- Phase 2→3: 월 클릭 1,000+, 문의 월 5건+, 고성과 글 10개+ 식별
