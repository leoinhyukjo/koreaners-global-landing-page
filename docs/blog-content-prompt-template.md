# 코리너스 블로그 콘텐츠 생성 프롬프트 템플릿

AI로 블로그 초안을 생성할 때 이 프롬프트를 사용하세요.

---

## 사용법

1. `[키워드]`와 `[필러 토픽]`을 실제 값으로 교체
2. Claude 또는 ChatGPT에 아래 프롬프트를 입력
3. 생성된 초안을 감수 체크리스트로 검토
4. 관리자 에디터에서 업로드

---

## 프롬프트

```
당신은 한-일 크로스보더 마케팅 전문가입니다. 코리너스(KOREANERS) 블로그에 게시할 글을 작성해주세요.

## 기본 정보
- 키워드: [키워드] (예: "일본 인플루언서 마케팅 비용")
- 필러 토픽: [필러 토픽] (예: "일본 인플루언서 마케팅")
- 타겟 독자: 한-일 크로스보더 비즈니스 관련 업계 관계자 (마케터, 경영진, 에이전시)
- 톤: 전문적이면서 읽기 쉬운 B2B 톤. 딱딱하지 않되 가볍지도 않게.

## 필수 구조 (이 순서를 반드시 따를 것)

### 1. Quick Answer (40-80자)
- 글의 첫 문장에서 키워드에 대한 직접 답변을 제공
- AI가 가장 먼저 인용하는 부분
- 예: "일본 마이크로 인플루언서(1만~10만 팔로워) 1회 시딩 비용은 5만~30만 엔(약 40만~240만 원) 수준이다."

### 2. 핵심 요약 박스
- 3-5개 bullet point
- 각 bullet에 구체적 수치나 핵심 팩트 포함
- 이 요약만 읽어도 글의 가치를 알 수 있어야 함

### 3. 본문 (H2 헤딩 3-5개)
- **모든 H2는 의문문으로** 작성 (AI가 Q&A 단위로 파싱)
  - 좋은 예: "일본 인플루언서 마케팅 비용은 얼마인가?"
  - 나쁜 예: "1. 일본 마케팅 전략 소개"
- 각 섹션 300-500자
- 최소 3개의 구체적 수치/통계 포함
- 비교표 최소 1개 포함 (마크다운 테이블)

### 4. FAQ (3-5개)
- 실제 고객이 물어볼 법한 질문
- 답변은 2-3문장으로 간결하게
- 검색 의도와 연관된 롱테일 키워드 포함

## 작성 규칙

- 글 전체 길이: 1,500-3,000자 (한국어 기준)
- 통계/수치에는 반드시 출처 표시 (연도 포함)
- "코리너스"를 본문에 1-2회 자연스럽게 언급 (강제 삽입 X)
- 마지막에 CTA 문구 제안: "일본 마케팅 전문가와 상담하고 싶다면 코리너스에 문의하세요."

## 출력 형식

다음 필드를 모두 포함해서 출력:

1. **제목** (40-60자, 키워드 포함)
2. **슬러그** (영문 소문자-하이픈, 예: japan-influencer-marketing-cost-guide-2026)
3. **카테고리** (업계 동향 / 최신 트렌드 / 전문가 인사이트 / 마케팅 뉴스 중 택 1)
4. **요약** (Quick Answer용, 40-80자)
5. **Meta Title** (50-60자, 검색 결과에 표시)
6. **Meta Description** (120-160자, 검색 결과 설명)
7. **본문** (위 구조대로)
8. **FAQ** (질문-답변 3-5쌍)
```

---

## 감수 체크리스트

초안 생성 후 반드시 확인:

- [ ] 수치/통계에 출처가 있는가? (연도 포함)
- [ ] Quick Answer가 40-80자로 핵심을 담고 있는가?
- [ ] H2가 모두 의문문인가?
- [ ] 비교표가 최소 1개 있는가?
- [ ] 코리너스 고유 인사이트가 1개 이상 포함되었는가? (실제 캠페인 경험 등)
- [ ] FAQ가 실제 고객이 물어볼 만한 질문인가?
- [ ] CTA가 자연스럽게 연결되는가?
- [ ] 글 길이가 1,500-3,000자인가?
- [ ] Meta Title 50-60자 / Meta Description 120-160자인가?
