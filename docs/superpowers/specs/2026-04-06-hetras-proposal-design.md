## 1. 배경 & 맥락

- 참석: 심보리 본부장(헤트라스) 외 1인 × 이동훈 부대표, 조인혁, 카오리(코리너스)
- 이전 모모나 타이업 회고 → 브랜드 인지는 됐으나 "뭘 사야 돼?" 개별 향 추천 부재로 매출 직결 안 됨
- 콜라보 > 앰버서더 > 타이업 선호 순서 확인

### 헤트라스 핵심 과제
1. **20대 타깃 확대**: 큐텐 30대 시장 성장률 둔화 → 20대 공략 필수
2. **바디 퍼스널케어 스큐 확장**: 바디미스트(2,000엔대) + 바디워시 → 디퓨저급 매출로 성장
3. **콘텐츠 전환**: 릴스 광고 피로도 → 스토리 중심 + 개별 향 딥 추천

### 원가 구조 제약
- 원가율 30~40% (뷰티 평균 10%)
- RS 30% 적용 시 마케팅비 포함하면 마이너스
- 대안: 가격 할인보다 기프트(증정품) 특전 + 대량 판매 유도

## 3. 섹션별 상세 설계

### 3.1 인사 & 미팅 요약
- 4/1 미팅 감사 인사
- 핵심 논의 사항 3줄 요약
- 목적: "우리가 정확히 이해했다"는 신뢰 구축

### 3.2 헤트라스 과제 정리
- 20대 타깃 확대
- 바디미스트·바디워시 스큐 확장
- 시향 없이 구매 전환하는 콘텐츠 필요
- 원가 제약에 따른 수익 구조 고려

### 3.3 코리너스 제안 개요

| | Option A: 콜라보(공구) | Option B: 앰버서더(장기) | Option C: 타이업(단일) |
|---|---|---|---|
| 목적 | 매출 즉각 전환 | 꾸준한 브랜드 침투 | 가볍게 시작/재개 |
| 기간 | 2~3개월 (메가와리 집중) | 6개월+ | 1회성 |
| 비용 레인지 | OOO~OOO만원 | 월 OOO~OOO만원 | OOO~OOO만원 |
| 핵심 크리에이터 | 모모나 | 모모나 + 마이크로 2~3명 | 모모나 or 신규 1명 |
| 콘텐츠 방향 | 스토리 중심 + 팬 참여형 | 일상 속 반복 노출 | 특정 향 딥 추천 |

### 3.4 Option A: 콜라보(공구) 패키지

**크리에이터**: 모모나 메인

**제품 포커스**: 바디미스트 (2,000엔대, 20대 트라이 가능)
- 지속시간 8시간+, 기능성(미백/진정), 5종 향 바리에이션
- 여름 시즌 궁합: 바디워시 레이어링 → 체향

**콘텐츠 전략**:
- 포맷: 스토리 중심 (릴스 광고 피로도 반영)
- 핵심: 특정 향 1~2종 딥 추천 ("모모나가 진짜 쓰는 향은 이거다")
- 팬 참여형: 인스타 유료 구독자 대상 향 설문 → 인기 향 시향 → 콘텐츠화
- "한국에 오면 꼭 사가는 오미야게" 포지셔닝

**수익 구조**:
- RS 대신 **고정 피 + 기프트 특전** 방식
- 원가율 30~40% 반영 → 추가 할인 없이 증정품으로 팬 특전 구성
- 대량 판매 유도로 객단가 확보

**비용 레인지**: OOO~OOO만원 (모모나 단가 + 기획·운영비)

**제약**: 룸투어 불가 (위치 특정 우려)

### 3.5 Option B: 앰버서더(장기) 패키지

**크리에이터**: 모모나(메인 앰버서더) + 마이크로 2~3명

**컨셉**: 일상 속 자연스러운 반복 노출
- "한국 거주 일본인의 집에 당연히 있는 제품"
- 6개월간 월 OO건 스토리/콘텐츠

**향 사전 히어링 프로세스**:
- 방법 1: 평소 쓰는 향수 → 노트 분석 → 맞춤 추천
- 방법 2: 싫어하는 향 제외 후 추천
- 심보리 본부장 제안 프로세스 그대로 적용

**크리에이터 매칭 기준**:
- 20대 타깃, 향/바디케어 카테고리 진성 관심
- 한국 거주 일본인 or 일본 현지
- 오가닉 톤 (광고 아닌 진짜 쓰는 느낌)

**비용 레인지**: 월 OOO~OOO만원

### 3.6 Option C: 타이업(단일) 패키지

**크리에이터**: 모모나 or 신규 1명

**이전 타이업 한계 개선**:
- 기존: 다양한 제품군 → "헤트라스 전체" 수준 인지 → 구매 직결 안 됨
- 개선: **특정 향 1종 딥 추천**으로 초점 좁히기 → "이거 사야 돼" 명확한 CTA

**포맷**: 스토리 1~2건 + 피드/릴스 1건

**비용 레인지**: OOO~OOO만원

### 3.7 크리에이터 라인업

**모모나 (메인)**:
- 프로필: YT 59.3만 / IG 35.7만 / TikTok 14.2만 / X 22.3만
- 포지셔닝: 한국 거주 10년+, 일상에 광고 자연스럽게 녹이는 인지 확대형
- 오디언스: 18~34세 여성 (YT 여성 91%, IG 70%)
- 레퍼런스: SKINFOOD 쿠폰 14,218매(사용률 56%), FOUNUA 라이브 150만엔, 시마무라 5일 2천착
- 헤트라스 이전 협업: 만족도 높음, DM 다수 수신

**추가 후보 2~3명**:
- 프로필 카드 형식: 채널명 / 팔로워 / 카테고리 / 오디언스 / 왜 이 크리에이터인지
- 선정 기준: 20대 타깃, 향·바디케어 진성, 한국 거주 일본인 or 일본 현지
- ※ 구체적 후보는 제안서 작성 시 Leo님과 확정

### 3.8 메가와리 연계 타임라인

6월 메가와리 역산, 3옵션 공통 적용:

| 시점 | 항목 |
|---|---|
| 4월 중순 | 옵션 확정 + 크리에이터 컨펌 |
| 4월 하순 | 향 사전 히어링 + 제품 발송 |
| 5월 초~중순 | 사전 콘텐츠 (브랜드 인지 웜업) |
| 5월 하순 | 메가와리 예고 콘텐츠 |
| 6월 | 메가와리 본 기간 — 매출 전환 집중 |
| 6월 말 | 성과 리뷰 + 후속 논의 |

### 3.9 추가 옵션: 바이럴 확산 부스터

- 메인 크리에이터 콘텐츠 게시 후, 마이크로 크리에이터 복수가 자체 콘텐츠로 2차 확산
- 오가닉 도달 극대화 — 알고리즘이 "트렌딩 주제"로 인식하는 효과
- 상세는 별도 협의 가능
